Chair transporting problem

Consider a chair manufacturing company with 2 factories (1, 2) and wants to transport them to retailers in 3 different cities (A, B, C). Supply at each factory (Table 1), demand at each retailer (Table 2), transport costs for one chair from each factory to each retailer (Table 3) are known.

Identify the unknowns (identify the decision variables)

The unknowns are the volume of chairs that is to be transported from each factory to each retailer.
X1A, X1B, X1C, X2A, X2B, X2C
Mathematical form: Xij, i={1,2}, j={A,B,C}

Formal expression including the unknowns (identify the objective function)

We can write the total cost of transporting the chairs to all the retailers in a mathematical form. We want to minimize this cost.
C(ost) = X1A*P1A + X1B*P1B + X1C*P1C + X2A*P2A + X2B*P2B + X2C*P2C
Mathematical form: C(ost) = sum (Xij*Pij); i = {1,2}, j = {A, B, C}

Business limitations imposed on the unknowns (identify the constraints)

The number of chairs transported to a particular retailer is at least equal with its demand.
The number of chairs transported from each factory has to be less than its production capacity.
X1A+X2A >= D_A; X1B+X2B >= D_B; X1C+X2C >= D_C; D_x is the demand by retailer x
X1A+X1B+X1C >= P1; X2A+X2B+X2C >= P2; Py is the production capacity of factory y

Practical limitations imposed on the unknowns (identify the non-negativity constraints)

The number of chairs transported from each factory to each retailer cannot be less than 0.
X1A >= 0, X1B >= 0, X1C >= 0, X2A >= 0, X2B >= 0, X2C >= 0

TThe optimal solution (Table 4) is X1A=55; X1B=0; X1C=29; X2A=0; X2B=31; X2C=49.
