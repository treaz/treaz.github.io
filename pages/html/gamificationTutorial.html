Level experience points (XP)

  1. Level 1: 0 XP
  2. Level 2: 50 XP
  3. Level 3: 160 XP

Chat room

This tutorial offers a very basic chat room in which you can communicate with other people going throught the tutorial. Note that when you will switch to a new page of the tutorial, you will lose the chat history.

This is not a game

The added game elements aim to make the experience more pleasurable, the intent of this tutorial is to teach you about optimization modeling.
